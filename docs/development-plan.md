# OS Baka 开发计划

## 项目概述

OS Baka 是一个具有 Apple 风格 UI 的 PXE 服务和物理资产管理系统，提供自动化加密安装功能。

## 核心功能需求

### 1. PXE 服务 (优先级：高)
- **网络启动服务**
  - TFTP 服务器配置
  - DHCP 服务器集成
  - iPXE/PXE 启动菜单
  - 启动镜像管理

- **操作系统部署**
  - macOS 网络安装支持
  - Linux 发行版支持
  - Windows PE 支持
  - 自定义安装脚本

### 2. Web UI 资产管理 (优先级：高)
- **资产信息管理**
  - IP 地址分配与追踪
  - MAC 地址注册与管理
  - 资产标签系统
  - USB Key 管理

- **资产操作**
  - 批量导入/导出
  - 资产搜索与过滤
  - 资产状态监控
  - 历史记录查询

### 3. 自动化加密安装 (优先级：中)
- **加密部署**
  - FileVault (macOS) 自动配置
  - BitLocker (Windows) 集成
  - LUKS (Linux) 支持
  - 加密密钥管理

- **安全配置**
  - 预配置密钥注入
  - 恢复密钥托管
  - 合规性检查

### 4. 节点管理 (优先级：中)
- **远程控制**
  - 节点重建/重装
  - 电源管理 (IPMI/AMT)
  - 启动顺序配置
  - 批量操作

- **监控与日志**
  - 部署进度追踪
  - 错误日志收集
  - 性能监控
  - 告警通知

### 5. Web Shell (优先级：低)
- **终端访问**
  - 基于 WebSocket 的 TTY
  - 多会话支持
  - 会话录制回放
  - 访问权限控制

### 6. SSH Key 管理 (优先级：低)
- **密钥管理**
  - 密钥对生成
  - 公钥分发
  - 密钥轮换
  - 访问审计

### 7. 部署支持 (优先级：高)
- **Docker 部署**
  - Dockerfile 编写
  - Docker Compose 配置
  - 镜像优化
  - 健康检查

- **Kubernetes 部署**
  - Helm Chart 开发
  - StatefulSet/Deployment 配置
  - Service/Ingress 配置
  - 持久化存储

## 技术栈选型

### 后端
- **框架**: FastAPI / Flask (Python)
- **数据库**: PostgreSQL / SQLite
- **缓存**: Redis
- **任务队列**: Celery / RQ
- **PXE 服务**: 
  - dnsmasq (DHCP + TFTP)
  - nginx (HTTP 文件服务)

### 前端
- **框架**: Svelte (参考 Apple App Store)
- **构建工具**: Vite
- **样式**: SCSS + CSS Variables
- **状态管理**: Svelte Stores
- **HTTP 客户端**: Axios

### 基础设施
- **容器**: Docker
- **编排**: Docker Compose / Kubernetes
- **反向代理**: Nginx / Traefik
- **监控**: Prometheus + Grafana

## 开发阶段

### 第一阶段：核心基础 (2-3周)
1. **项目初始化**
   - 创建项目结构
   - 配置开发环境
   - 设置 CI/CD 管道

2. **数据库设计**
   - 设计数据模型
   - 创建迁移脚本
   - 编写 ORM 模型

3. **基础 API**
   - 资产 CRUD API
   - 认证与授权
   - API 文档 (OpenAPI)

4. **Apple 风格 UI 框架**
   - 提取和适配设计系统
   - 创建基础组件库
   - 实现响应式布局

### 第二阶段：资产管理 (2-3周)

   - 资产标签系统

   - 导入/导出功能

2. **镜像管理**
   - 镜像上传与存储

3. **部署流程**
